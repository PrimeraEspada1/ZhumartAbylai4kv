Домашняя страница

Веб-сервис аутентификации для сайта по ремонту компьютеров

Эта часть проекта сделана для лабораторной работы на Django: здесь есть главная страница, регистрация, вход, база данных пользователей, админ-панель и система загрузки фото профиля.

Что реализовано

Функциональность проекта

  • Регистрация пользователя через форму Django.
  • Вход и выход через систему аутентификации.
  • Профиль пользователя с личной фотографией.
  • Хранение фото в отдельных папках вида avatars/user_ID/.
  • Ограничение на загрузку только изображений.
  • Поддержка фото по умолчанию, если загрузки ещё не было.
